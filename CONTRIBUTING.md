Contributing to Pipenv
======================

Please see: [docs](https://pipenv.pypa.io/en/latest/dev/contributing.html).

Before editing code under `pipenv/`, note in particular the
["Owned code vs. vendored code" policy](https://pipenv.pypa.io/en/latest/dev/contributing.html#owned-code-vs-vendored-code):
code is either project-owned (refactor freely) or vendored (do not
hand-edit; the vendoring tooling re-syncs from upstream). There is no
third state.
